DBMS

Wat is data?

Het woord data is de meervoudsvorm van het woord ‘datum’, dat ‘één stukje informatie’ betekent.
Data is informatie die wordt gebruikt en opgeslagen door de computer.
Data heeft verschillende vormen. Het kan tekst zijn, maar ook foto’s, filmpjes en zelfs softwareprogramma's. Deze data wordt verwerkt door de CPU en staat opgeslagen op de harde schijf van je computer.
De data staat opgeslagen als binary data (reeks van 1 en 0).

Wat is database?

Database is een systematische verzameling van data. Ze ondersteunen elektronische opslag en manipulatie van data. Databases maken databeheer eenvoudig. Man kan data organiseren in tabellen, rijen, kolommen en indexen om het gemakkelijker te maken om relevante informatie te vinden. Handlers maken een database op zo'n manier dat slechts één set softwareprogramma toegang biedt tot data voor alle gebruikers. Het hoofddoel van de database is om een ​​grote hoeveelheid informatie te beheren door het opslaan, ophalen en beheren van data. Er zijn tegenwoordig veel dynamische websites op the world wide web , die worden beheerd via databases. Bijvoorbeeld een model dat de beschikbaarheid van kamers in een hotel controleert. Het is een voorbeeld van een dynamische website die een database gebruikt.

Wat is DBMS?

DBMS (Data management system) is een software die het maken, definiëren en manipuleren van databases mogelijk maakt, waardoor gebruikers processen en data eenvoudig kunnen opslaan en analyseren. DBMS biedt een interface of een tool om verschillende bewerkingen uit te voeren, zoals het maken, opslaan, updaten van data, het maken van tabellen in de database en nog veel meer. DBMS biedt ook bescherming en beveiliging voor de databases. Het handhaaft de consistentie van de data in het geval van meerdere gebruikers.

Kenmerken DBMS

Het slaat data op in tabellen.
Data wordt nooit rechtstreeks in de database opgeslagen. Data wordt opgeslagen in tabellen die in de database zijn gemaakt.

DBMS maakt het ook mogelijk om relaties tussen tabellen te hebben, wat de data betekenisvoller en meer verbonden maakt. Man kan eenvoudig begrijpen welk type data waar is opgeslagen, door te kijken naar alle tabellen die in een database zijn gemaakt. In de moderne wereld zijn harde schijven erg goedkoop, maar vroeger, toen harde schijven te duur waren, was onnodige herhaling van gegevens in de database een groot probleem, maar dbms volgt normalisatie die de gegevens op zo'n manier verdeeld dat herhaling minimaal is.

Ondersteunt meerdere gebruikers en gelijktijdige toegang.
Dbms staat meerdere gebruikers toe om er tegelijkertijd aan te werken en weet toch de datum consistentie te behouden: er zijn voornamelijk vier bewerkingen die op de database worden uitgevoerd. Deze bewerkingen worden CRUD-bewerkingen genoemd (create, retrieve, update and delete).

Query taal.
DBMS biedt gebruikers een eenvoudige query taal waarmee gegevens eenvoudig kunnen worden opgehaald, ingevoegd, verwijderd en bijgewerkt in een database. Een voorbeeld van een querytaal is SQL.

Beveiliging.
Het DBMS zorgt ook voor de beveiliging van gegevensbescherming. De gegevens tegen ongeautoriseerde toegang in een typische DBMS kunnen we gebruikersaccounts maken met verschillende toegangsrechten waarmee we onze gegevens eenvoudig kunnen beveiligen door de toegang van gebruikers te beperken. Enkele populaire databasebeheersystemen zijn MySQL, Microsoft Access, PostgreSQL, Oracle, SQLite en MariaDB
